試題反應理論（IRT）模型詳細介紹

1PL模型 (Rasch Model)

1PL模型，也被稱為Rasch模型，僅考慮一個參數，即試題的難度。在此模型中，所有試題都被假定具有相同的鑑別度。

  • 試題難度： 它描述了當受測者的能力為零時，該試題被正確回答的機會。
  • 機率計算公式（表達式1）：P(Y_{ij}=1) = 1 / (1 + exp(-(\theta_j - b_i)))
  • 機率計算公式（表達式2）：\[ P(Y_{ij}=1) = \frac{1}{1 + \exp(-(\theta_j - b_i))} \]

2PL模型 (Two-Parameter Logistic Model)

2PL模型包含兩個參數：試題的難度和鑑別度。鑑別度參數能夠衡量試題區分不同能力水平受測者的效果。

  • 試題難度： 同1PL模型中的定義。
  • 鑑別度： 表示試題區分不同能力水平的能力。一個高鑑別度的試題對於能力較高和較低的受測者有很好的區分度。
  • 機率計算公式（表達式1）：P(Y_{ij}=1) = c_i + (1-c_i) * 1 / (1 + exp(a_i(\theta_j - b_i)))
  • 機率計算公式（表達式2）：\[ P(Y_{ij}=1) = c_i + (1-c_i) \times \frac{1}{1 + \exp(a_i(\theta_j - b_i))} \]

3PL模型 (Three-Parameter Logistic Model)

3PL模型在2PL模型的基礎上增加了猜測參數，它表示當受測者的能力值非常低時，其猜測該試題正確答案的機會。

  • 猜測參數： 表示當受測者的能力極低時，他們依然有機會猜對答案。此參數確保了正確回答的機率永遠不會降到0。
  • 機率計算公式（表達式1）：P(Y_{ij}=1) = c_i + (1-c_i) * 1 / (1 + exp(a_i(\theta_j - b_i)))
  • 機率計算公式（表達式2）：\[ P(Y_{ij}=1) = c_i + (1-c_i) \times \frac{1}{1 + \exp(a_i(\theta_j - b_i))} \]

4PL模型 (Four-Parameter Logistic Model)

4PL模型在3PL模型的基礎上引入了上限參數，該參數描述了當受測者的能力遠高於試題難度時，其正確回答的機會的上限。

  • 上限參數： 表示試題回答正確的機會上限。當受測者能力極高時，其正確回答的機會不會超過此上限。
  • 機率計算公式（表達式1）：P(Y_{ij}=1) = c_i + (d_i-c_i) * 1 / (1 + exp(a_i(\theta_j - b_i)))
  • 機率計算公式（表達式2）：\[ P(Y_{ij}=1) = c_i + (d_i-c_i) \times \frac{1}{1 + \exp(a_i(\theta_j - b_i))} \]

5PL模型 (Five-Parameter Logistic Model)

5PL模型在4PL模型的基礎上引入了不對稱參數，它能夠控制機率曲線的不對稱性，使其更符合實際情況。

  • 不對稱參數： 控制機率曲線的不對稱性。通常用於模擬特定情況下的試題反應，如測試兩極端能力的受測者。
  • 機率計算公式（表達式1）：P(Y_{ij}=1) = c_i + (d_i-c_i) * 1 / (1 + exp(a_i(\theta_j - b_i)e_i))
  • 機率計算公式（表達式2）：\[ P(Y_{ij}=1) = c_i + (d_i-c_i) \times \frac{1}{1 + \exp(a_i(\theta_j - b_i)e_i)} \]
